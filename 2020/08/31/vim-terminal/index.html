vim && terminal

发表于 2020-08-31 更新于 2021-01-08 分类于 linux

最近看到了一篇关于vim的学习曲线文章《简明 VIM 练级攻略》，才发现自己也就比第一级强了那么一点点:(

最初接触vi/vim时，最多在linux下修改配置文件的几个参数，没有特意去记忆、使用更多的快捷键。

使用linux多了，比如远程修改nginx服务器的配置文件，渐渐的发现需要掌握更多的vim操作快捷键。

在linux中，例如man指令，阅读帮助文档时的很多快捷键，和vim快捷键一致。

vi/vim

在《简明 VIM 练级攻略》中给了一种学习曲线，由易到难，可以克服初学者的恐惧心里。

看了vi/vim graphical cheat sheet中的键盘标记，针对电脑上的文本编辑操作，可以分几类记忆：

  • 光标移动 motion
    • 方向键：h，j，k，l
    • 文本行内移动光标：w，e，b，0，$
    • 句子，段落移动：（），{}
    • 移动到指定行号：gg（文档首行），G（文档末尾），ngg（移动到第n行）
    • 查找：fx（行内查找字符x并移动），/<表达式> （全文查找，多个查找结果使用n，N切换下一个）
    • mark标记：ma…mz（标记a…z）,’a…’z（），`a…`z（移动到a…z）
    • 翻页：ctrl+b/f
  • 编辑 command
    • 进入插入模式（insert mode）：i，a，o（对应大写I，A，O）
    • 复制/粘贴：y/p（yy复制行；复制操作y，结合w，e，b，0，$等对应移动光标操作，可以复制光标移动范围内容）
    • undo/redo：u/ctrl+r
    • 块操作：ctrl+v
    • 删除：d （dd删除行；删除操作d，结合w，e，b，0，$等对应移动光标操作，可以删除光标移动范围内容）

linux terminal

linux最多使用的是terminal工具，特别是通过ssh远程连接到linux服务器进行管理。

虽然每次命令行只有一行，但使用时，终端提供了很多快捷键(默认的是emacs模式)，通过set -o vi指令，可以将终端快捷键模式修改为vi快捷键，此时终端进入的是vi insert mode，需要编辑命令行时，按ESC键，进入command mode，此时就可以使用vi的快捷键移动光标。

考虑到大多数的默认配置是emacs模式，还是记录该模式快捷键比较好，下面是一些常用快捷键：

快捷键 说明
Tab 按一次补全指令或路径，按两次列出所有相关信息。
Shift + Insert 从系统剪贴板粘贴，不同终端可能有其它快捷键
Ctrl + l 清屏

行内移动光标

快捷键 说明
Ctrl + a 移动光标到命令行首。
Ctrl + e 移动光标到命令行尾。
Alt + f
Ctrl + ⬅		 向前移动一个单词
Alt + b
Ctrl + ➡		 向后移动一个单词
Alt + a 将光标移到当前单词头部
Alt + e 将光标移到当前单词尾部

命令行编辑

快捷键 说明
Ctrl + k 剪切文本直到行的末尾，删除光标后面所有字符
Ctrl + u 剪切文本直到行的起始，删除光标前面所有字符
Ctrl + w 剪切光标前的单词(光标位置到前方空格或行首)
Ctrl + y 粘贴上面指令删除内容
Ctrl + d 删除当前的字符
Ctrl + t 交换当前字符和前一个字符的位置
Alt + t 交换当前单词和前一个单词的位置
Alt + u 把当前单词变成大写
Alt + l 把当前单词变成小写
Alt + c 把当前单词变成首字母大写的单词
Alt + d 删除从光标到当前单词结尾的部分
Alt + y 回退到先前剪切的文本并粘贴它
Ctrl + x + u 撤销刚才的操作
Ctrl + c 另起新行

程序相关

快捷键 说明
Ctrl + z 中断程序放到后台，唤醒使用“fg”命令。
Ctrl + s 挂起，暂停屏幕输出，感觉类似于暂停。
Ctrl + q 退出挂起，感觉类似于继续，发现按键盘其它按键也可以退出挂起，有点疑问。
Ctrl + c 中断并杀死，程序终止。
Ctrl + d “EOF” (文件结尾：end of file)。它用于表示标准输入（stdin）的结束。
例如：进入python交互式解释器或ssh远程登陆后，按Ctrl+d键退出
在控制台或xterm 窗口输入文本时， 删除在光标下的字符。

快捷键 说明
Ctrl + r 搜索匹配最近使用的指令（）

Ctrl + n 显示下一条命令，同向下箭头。
!$ 显示系统最近的一条参数，应用例子：
先执行cat /etc/sysconfig/network-scripts/ifconfig-eth0，然后用vim编辑。
一般做法：先↑ 显示最后一条命令，Home移到最前，删除cat，输入vim。其实可用vim !$代替
