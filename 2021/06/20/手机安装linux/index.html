手机安装linux

发表于 2021-06-20 更新于 2021-06-27 分类于 linux ， 手机软件

最近的就手机上想安装linux，手机不能root,不能安装linux deploy（参考：手机安装LinuxDeploy 使用总结 2019），本想直接安装Termux，发现了一个新的手机端linux软件aid learning……

Termux 需要android7以上版本，aid learning要求更高，支持鸿蒙系统，手里有个手机android版本较低，只能使用linux deploy

Termux

初始安装

# 修改清华镜像
sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list

sed -i 's@^\(deb.*games stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable@' $PREFIX/etc/apt/sources.list.d/game.list

sed -i 's@^\(deb.*science stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science stable@' $PREFIX/etc/apt/sources.list.d/science.list

# 修改root密码
passwd

# 安装
pkg install openssh

# 启动sshd 
## 没有root，默认端口8022，ssh的默认端口22不能使用
sshd

## 后面可以通过电脑ssh客户端连接
echo "sshd" >> ~/.bashrc

apt update && apt upgrade

# 常用软件
termux-setup-storage # 本地存储
pkg install python # 安装python3
pkg install vim-python
pkg install nginx

aid learning

meefik/linuxdeploy: Install and run GNU/Linux on Android (github.com)
