supervisor && rc.local

linux deploy 中不能使用systemctl进行后台服务管理，可以用supervisor代替，例如frp后台启动管理：

apt install supervisor

vim /etc/supervisor/conf.d/frp.conf

# frp.conf
[program:frps]
command =/usr/local/frp/frps -c /usr/local/frp/frps.ini
autostart = true

安装Linux时，选择了“初始化-启用-允许使用初始化系统”，可以再/etc/rc.local中增加启动脚本

注：需要给rc.local增加权限chmod 777 /etc/rc.local
